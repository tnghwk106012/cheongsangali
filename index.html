왜 연막 5문인가?

장점
  • ✓ 연막 다수 장착 시 특수연막탄 측타의 스킬 해제 확률 상승
  • ✓ 사정거리가 길어 전선 깊숙이 진입 불필요 → 생존 유리
  • ✓ 장전속도 빠름 (포술 25랭 = 4틱, 5틱 대비 체감 차이 큼)
  • ✓ 뉴비 단계에서 솔킬보다 스킬 해제 1회가 팀 기여도 훨씬 높음
유탄포가 뉴비 비추인 이유
  • ✗ 장전속도가 느려 개특슈/팬텀과 틱이 안 맞음
  • ✗ 속사로 틱 보정하거나 장전 게이지를 보며 쏴야 함
  • ✗ 두 방법 모두 뉴비에겐 상당한 부담
대포틱이란? 화면 우하단 대포 게이지가 올라가는 눈금 단위. 포술 25랭 달성 시 4틱이 되며, 5틱과 실전 체감 차이가 상당함.
핵심 요약: 강포전은 90% 이상 돌대안개 메타. 뉴비가 혼자 킬을 내기 어려우므로, 연막탄으로 적 스킬 해제에 집중하는 것이 최선의 팀 기여.

+1 스킬 비교: 왜 백요인가?

스킬뉴비 적합도이유
백요 최추천 외과 부담 감소. 전장에서 특구명 쓸 여유 없을 때 자동 회복. 파티원에게 외과 반복 요청하는 부담도 줄어듦
장개 보통 생존에 좋지만, 현 메타(3~4명 집중 타겟팅)에선 장개가 있어도 빠르게 녹음. 뉴비는 우선 타겟팅 대상이 되기 쉬움
굉기 비추 전장이 정신없어서 기뢰 1개 깔기도 힘듦
백회 비추 위급 시 백병 걸어 시간 버는 것조차 불가능 → 오히려 더 빨리 죽음
뉴비 핵심 원칙: 손이 덜 가는 세팅이 최우선. 스스통+백요는 가격도 저렴하고 나중에 부관배/모전배로 전용 가능.

• 평일에 쟁이 없을 때 할 것이 없음
• 주말 참여 → 뉴비라 먼저 죽고 → 다음 주말 대기 → 반복

• 이렇게 접는 분들을 많이 봄
